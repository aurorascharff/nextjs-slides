# nextjs-slides demo

A minimal demo app showcasing the nextjs-slides library. Matches the original [nextjs-demo-kit](https://github.com/aurorascharff/nextjs-demo-kit) slide deck.

**[Live demo â†’](https://nextjs-slides.vercel.app/)**

## Run

From the repo root:

```bash
npm run build          # build the library first
cd examples/demo
npm install
npm run dev
```

Open http://localhost:3000 and click "Open slides" to start the deck.
