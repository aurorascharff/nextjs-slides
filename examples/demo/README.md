# nextjs-slides demo

A minimal demo app showcasing the nextjs-slides library. Matches the original [nextjs-demo-kit](https://github.com/aurorascharff/nextjs-demo-kit) slide deck.

**[Live demo →](https://nextjs-slides.vercel.app/)**

## Run

From the repo root:

```bash
npm run build          # build the library first
cd examples/demo
npm install
npm run dev
```

Open http://localhost:3000 and choose a deck:

- **Geist deck** (`/slides`) — Default theme, Geist fonts. Full primitives showcase.
- **Alternate deck** (`/slides-alt`) — Playfair Display + JetBrains Mono, Dracula-inspired syntax.
